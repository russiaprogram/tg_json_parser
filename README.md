# Telegram JSON Parser

Parse Telegram JSON export files into structured data and CSV format.

## Overview

This module parses JSON files exported from Telegram Desktop and converts chat messages into a structured format for analysis. **Zero external dependencies** - uses only Python standard library.

## Features

- Parse Telegram JSON exports
- Extract message content and metadata
- Handle different message types (text, media, polls, locations)
- Export parsed data to CSV
